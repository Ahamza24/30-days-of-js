## Remove Duplicates

Write a function that takes an array of integers `arr` and returns a new array with duplicates removed, preserving the original order.

### Example 1
- **Input:** `arr = [1, 2, 2, 3, 4, 4, 5]`
- **Output:** `[1, 2, 3, 4, 5]`

### Example 2
- **Input:** `arr = [5, 5, 5, 5]`
- **Output:** `[5]`
