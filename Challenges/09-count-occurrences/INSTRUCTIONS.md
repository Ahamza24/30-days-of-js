## Count Occurrences

Write a function that takes an array of integers `arr` and an integer `x`, and returns the number of times `x` appears in the array.

### Example 1
- **Input:** `arr = [1, 2, 2, 3, 4], x = 2`
- **Output:** `2`

### Example 2
- **Input:** `arr = [5, 5, 5, 5], x = 5`
- **Output:** `4`
